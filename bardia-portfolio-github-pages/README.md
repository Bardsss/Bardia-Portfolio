# Bardia Jahanbakhsh Portfolio

Static GitHub Pages portfolio site for IT security, automation, PCL contributions, and cybersecurity research.

## Deploy with GitHub Pages

Best repo name for a personal site:

```text
Bardsss.github.io
```

Then upload these files to the root of the repo:

- `index.html`
- `styles.css`
- `script.js`

Go to:

```text
Settings → Pages → Deploy from branch → main → /root → Save
```

Your site will be available at:

```text
https://Bardsss.github.io
```

Alternative project repo name:

```text
bardia-portfolio
```

URL would be:

```text
https://Bardsss.github.io/bardia-portfolio
```
